# `zde rebuild`

`rebuild` exists as a reserved host-only command, but it is not implemented in the current wrapper.

## Usage

```sh
zde rebuild
```

## Current Behavior

- Prints a message that the host-only command is pending reimplementation.
- Exits with an error.

## Status

Use `zde update` and the regular build commands instead. There is no supported `rebuild` workflow in the current codebase.
